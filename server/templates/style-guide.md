# Awedan Sahayak — Formal Hindi Legal Application Style Guide

This document defines the standard structure for all government
applications (आवेदन पत्र) generated by Awedan Sahayak. The Claude
API must follow this format exactly for every generated application.

---

## Structural Template

Every application must use the following 7-part structure:

### 1. HEADER — सेवा में (To)

```
सेवा में,
{{designation}},
{{office_name}},
थाना/अंचल–{{police_station_or_block}},
जिला–{{district}},
राज्य–{{state}}।
```

- `{{designation}}` is derived from the office type:
  - Thana → "थाना प्रभारी/थानाध्यक्ष महोदय"
  - Block/Tehsil → "तहसीलदार/ब्लॉक अधिकारी महोदय"
  - BDO → "खंड विकास अधिकारी महोदय"
  - CO → "सर्किल अधिकारी/राजस्व अधिकारी महोदय"
  - SDO → "अनुविभागीय अधिकारी महोदय"
  - SP → "पुलिस अधीक्षक महोदय"
  - DC/DM → "जिलाधिकारी/जिला दंडाधिकारी महोदय"
  - Court → "माननीय न्यायाधीश महोदय"
- Use `थाना–` for police matters, `अंचल–` for revenue/block matters.

### 2. SUBJECT — विषय

```
विषय: {{subject_summary}}
```

One complete sentence that summarizes BOTH the issue AND the
specific action being requested. Examples:
- "चोरी की घटना के संबंध में प्राथमिकी दर्ज कर अभियुक्तों के विरुद्ध
  विधिक कार्रवाई करने हेतु प्रार्थना पत्र।"
- "गुमशुदा व्यक्ति की तलाश हेतु एवं प्राथमिकी दर्ज करने हेतु
  प्रार्थना पत्र।"

### 3. OPENING — महोदय / सविनय निवेदन

```
महोदय,

सविनय निवेदन है कि मैं {{applicant_name}}, {{parent_type}}
{{parent_name}}, ग्राम–{{village}}, थाना–{{police_station}},
जिला–{{district}} {{gender_residence_term}} हूँ।
```

- `{{parent_type}}` = "पिता" or "पति" based on applicant gender
  and marital context. Default to "पिता" if unknown.
- `{{gender_residence_term}}` must match applicant gender:
  - Male (पुरुष) → "का निवासी"
  - Female (महिला) → "की निवासिन"
- For court applications, use the deponent/petitioner name instead
  of "मैं."

### 4. BODY — घटना का विवरण (Narrative)

This is the most important section. The AI must:

- **Write in full narrative paragraphs**, NOT bullet points or
  numbered facts
- Describe the incident **chronologically** (what happened first,
  then next, then after that)
- Include **accused persons' full names WITH their father's/husband's
  names and village** (e.g., "गाँव के ही रहने वाले {{name1}}
  पिता {{father1}}, {{name2}} पिता {{father2}} ने...")
- Mention **specific dates, times, and locations** woven into the
  narrative
- Describe **what exactly happened**, including any weapons used,
  injuries sustained, witnesses present, and police/authority
  response (if any)
- Keep the tone **respectful, factual, and formal** — use "उक्त
  व्यक्तियों द्वारा" not "उन्होंने", use "प्रार्थी" or "आवेदक" not
  "मैंने"
- Length: typically 200–400 words, depending on incident complexity

Example narrative style (NOT to be copied verbatim — this is tone
reference only):

> "दिनांक {{date}} को लगभग {{time}} बजे, जब प्रार्थी अपने घर पर
> था/थी, तभी गाँव के ही रहने वाले {{accused1}} पिता {{father1}},
> {{accused2}} पिता {{father2}} एवं 2-3 अज्ञात व्यक्ति प्रार्थी के
> घर पर आ धमके और गाली-गलौज करने लगे। जब प्रार्थी ने विरोध किया तो
> उक्त सभी व्यक्तियों ने प्रार्थी के साथ लाठी-डंडों से मारपीट की,
> जिससे प्रार्थी के सिर एवं बाएँ हाथ में गंभीर चोटें आई हैं। घटना
> के समय प्रार्थी का/की {{relation}} {{witness_name}} भी मौजूद
> थे/थीं जिन्होंने बीच-बचाव का प्रयास किया। घटना के पश्चात प्रार्थी
> ने तत्काल {{hospital_name}} में जाकर अपना इलाज कराया, जिसकी
> मेडिकल रिपोर्ट संलग्न है।"

### 5. CLOSING REQUEST — अतः श्रीमान/महोदय से निवेदन

```
अतः श्रीमान/महोदय से सविनय/विनम्र निवेदन है कि उपरोक्त घटना के
संबंध में {{specific_action_requested}} की जाए, ताकि प्रार्थी को
न्याय प्राप्त हो सके एवं भविष्य में ऐसी घटनाओं की पुनरावृत्ति न हो।
```

The `{{specific_action_requested}}` must be:
- **Specific and actionable** — what EXACTLY should the officer do?
  (e.g., "प्राथमिकी दर्ज कर अभियुक्तों के विरुद्ध विधिक कार्रवाई",
  "उचित जाँच कर आवश्यक प्रमाण पत्र निर्गत करने की कृपा",
  "राजस्व अभिलेखों में संशोधन कर नवीन खतौनी निर्गत करने")
- **Formal Hindi** — use "की जाए", "किया जाए", "प्रदान किया जाए",
  not "करें", "दें"
- Use `श्रीमान` for male-designated officers, `महोदय` as
  gender-neutral fallback

### 6. GRATITUDE — आभार

```
इसके लिए मैं सदैव {{gender_gratitude_term}} रहूँगा/रहूँगी।
```

- `{{gender_gratitude_term}}`:
  - Male officer addressed → "आपका आभारी"
  - Female officer addressed → "आपकी आभारी"
  - `रहूँगा` for male applicant, `रहूँगी` for female applicant

### 7. FOOTER — हस्ताक्षर

```
दिनांक: {{date_today}}
स्थान: {{place}}

                            भवदीय/भवदीया,

                            {{applicant_name}}
                            {{parent_type}} {{parent_name}}
                            ग्राम–{{village}},
                            थाना–{{police_station}},
                            जिला–{{district}}
                            मोबाइल: {{applicant_phone}}
```

- `भवदीय` for male applicant, `भवदीया` for female applicant
- The place name should match the village/city where the applicant
  is signing
- Date format: "दिनांक XX माह 20XX" (Hindi-style date)

## Gender-Aware Grammar Rules

The `gender` field (`पुरुष` / `महिला` / `अन्य`) controls these
grammatical inflections throughout the entire document:

| Context | Male (पुरुष) | Female (महिला) |
|---------|-------------|----------------|
| Residence | का निवासी | की निवासिन |
| Gratitude | आपका आभारी | आपकी आभारी |
| I will remain | रहूँगा | रहूँगी |
| Valediction | भवदीय | भवदीया |
| I am | मैं ... हूँ | मैं ... हूँ (same) |
| My (poss.) | मेरा | मेरी |
| Applicant was | प्रार्थी ... था | प्रार्थी ... थी |
| Parent indicator | पिता | पति (if married) / पिता |

For `अन्य` gender, default to masculine forms but use the
gender-neutral `प्रार्थी` throughout rather than gendered pronouns.

## Narrative Quality Guidelines

The AI must:

1. **Synthesize, don't list** — take the discrete form fields
   (`incident_details`, `accused_names`, `injury_details`, etc.)
   and weave them into a single flowing narrative paragraph

2. **Include all names with identifiers** — every person mentioned
   must have their father's/husband's name and village attached on
   first mention

3. **Use formal legal vocabulary** — "प्रार्थी" (not मैं),
   "उक्त/उपरोक्त" (not वह), "अभियुक्तगण" (not वे लोग),
   "संलग्न" (not साथ में)

4. **Write complete sentences** — no fragments, no bullet points, no
   numbered lists in the narrative section

5. **Match the exemplar tone** — respectful, deferential to
   authority, factual but with appropriate urgency

6. **Never invent facts** — only narrate details that were provided
   in the form fields. If a detail is missing, omit it rather than
   fabricating it
